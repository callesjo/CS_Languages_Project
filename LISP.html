LISP Programming Language

It was from the mathematical theory of recursive functions that LISP was created around 1960, making it the second oldest programming language. It was developed by John McCarthy at Massachusetts Institute of Technology (MIT). LISP uses the so called list structure to represent data which makes it easy for it to operate on other programs as data since programs and data use the same structure.
  LISP was created as a practical mathematical notation for computer programs, but it became a common language for AI programming as it allows programmers to create programs that can think and learn. LISP has, among other things, been used to create programs called expert systems, that copies the decision-making process of human experts, as well as natural language processing, the ability of computers to understand human language.

    LISP is used for web development, database management, system administration and artificial intelligence. Here are some examples of programs and applications where LISP is used:
